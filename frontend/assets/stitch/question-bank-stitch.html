Easy Strings

Given a string, write a function to check if it is a palindrome after removing all non-alphanumeric characters and ignoring cases.

A. O(n) Time, O(1) Space
B. O(n log n) Time, O(n) Space
C. O(n²) Time, O(1) Space
D. O(n) Time, O(n) Space
SHOW ANSWER & EXPLANATION expand_more

Correct Answer: A

The most efficient way is to use two pointers (left and right). Move them towards the center, skipping non-alphanumeric characters. Compare characters at each step. This ensures we only traverse the string once (O(n)) and use no extra space besides a few pointers (O(1)).

Hard Dynamic Programming

Given an array of non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.

A. O(2ⁿ) Recursion
B. O(n) Two Pointers
C. O(n²) Nested Loops
D. O(n log n) Sort & Greedy
SHOW ANSWER & EXPLANATION expand_more

Correct Answer: B

While Dynamic Programming (memoization) works in O(n), the Two-Pointer approach is generally preferred as it reduces space complexity from O(n) to O(1).
